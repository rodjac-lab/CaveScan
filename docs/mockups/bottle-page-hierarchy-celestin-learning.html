Lecture du vin

Texte editorial principal visible sur la fiche. Il peut venir de Celestin mais doit rester corrigeable par l'utilisateur.

Personnel

Champ libre, prive, personnel. Visible ensuite sur la fiche si non vide, mais toujours distinct du texte editorial.

Un seul texte principal visible : le texte editorial remplace le flou autour de `character`.
Les notes perso restent : elles existent seulement comme complement, jamais comme doublon du texte principal.
Les deux champs sont editables : l'utilisateur peut corriger l'IA sans perdre sa propre voix.
